# What Light Remains

A digital installation that creates colors shaped by your time of day.

![What Light Remains](public/og.png)

[**View Live →**](https://www.whatlightremains.com/)

Inspired by the works of [Tori Hinn](https://www.instagram.com/torihinn/) and ["Remember Me as a Time of Day"](https://music.apple.com/nz/song/remember-me-as-a-time-of-day-remastered/318867418) by Explosions in the Sky.

## Built with

- WebGL and a fragment shader
- TypeScript
- Vite
- CSS
