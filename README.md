# What Light Remains

A digital art installation that generates colors shaped by your time of day.

[**View Live →**](https://www.ofatime.com/)

Inspired by the works of [Tori Hinn](https://www.instagram.com/torihinn/) and "Remember Me as a Time of Day" by Explosions in the Sky.

## Built with

- WebGL and a fragment shader
- TypeScript
- Vite
- CSS
