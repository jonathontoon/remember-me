# What Light Remains

A digital art installation that generates colors shaped by your time of day.

[**View Live →**](https://www.ofatime.com/)

Inspired by "Remember Me as a Time of Day" by Explosions in the Sky and the works of Tori Hinn.

## Built with

- WebGL and a fragment shader
- TypeScript
- Vite
- CSS
