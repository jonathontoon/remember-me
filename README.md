# What Light Remains

A digital installation that creates colors shaped by your time of day.

![What Light Remains](public/og.png)

[**View Live →**](https://www.whatlightremains.com/)

Inspired by the works of [Tori Hinn](https://www.instagram.com/torihinn/) and the song ["Remember Me as a Time of Day"](https://music.apple.com/nz/song/remember-me-as-a-time-of-day-remastered/318867418) by Explosions in the Sky.

## Built with

- WebGL and a fragment shader
- TypeScript
- Vite
- CSS

## URL parameters

- `time=HH:MM` sets the time of day, for example `?time=18:30`.
- `date=YYYY-MM-DD` sets the date that seeds the daily composition, for example
  `?date=2026-07-13`.
- The parameters can be combined, for example
  `?date=2026-07-13&time=18:30`.
